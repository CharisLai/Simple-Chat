## 簡易聊天室
簡介：使用socket.io 函式庫，做到Server監聽多個client間的訊息傳遞與狀態

### 預先安裝
- node.js 16.x

### 使用說明
1. Clone 至指定檔案位址
```
git clone https://github.com/CharisLai/Simple-Chat.git
```
2. 安裝
```
npm install
```
3. 運行
```
npm run dev
```
4. 打開瀏覽器，輸入：
```
localhost:3000
```
5. 請到 terminal 按下  ` ctl + c `結束server運作
